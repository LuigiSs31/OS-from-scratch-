
# Basic Operating System
## A Basic x86 Operating System



This repository contains the source code for a minimal, "from-scratch" x86 operating system. This was developed as the final project for the Operating Systems class at Universidad Autonoma de Guadalajara.

The primary goal of this project was not to create a full-featured OS, but to learn and demonstrate the fundamental concepts of how an operating system boots and interacts with hardware.
